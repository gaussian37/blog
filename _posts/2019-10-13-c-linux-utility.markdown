## **terminator**

- 우분투를 사용할 때 터미널을 분할 하시고 싶을 때가 간간히 있습니다. 이럴 때 유용한 프로그램입니다. 
- 먼저 terminator를 설치합니다. 
    - `sudo apt install terminator`
- 터미널에 `./terminator`라고 치면 사용할 수 있습니다.
- terminator 를 분할하는 핫 키는 다음과 같습니다.
    - ctrl + shift + e 세로분할
    - ctrl + shift + o 가로분할
    - ctrl + shift + w 닫기
    - ctrl + shift + q 종료
